
Setup instructions
==================

The default dir hierarchy looks like this:

data_dir               = /spare/tmp/tabled/lib
tdb_dir                = /spare/tmp/tabled/lib/tdb
pid_file               = /spare/tmp/tabled/run/tabled.pid


*) Create/choose location for process-id file.  Pass this to the
   daemon using the argument --pid=FILE

*) Create/choose data directory, where sqlite metadata and object data
   is stored.  Pass this to the daemon using the argument --data=DIRECTORY

*) Initialize the database inside the data directory:

	sqlite3 $data_dir/master.db < mk-tabledb.sql

   The schema file mk-tabledb.sql is in the tarball with the rest of the
   source code.

*) Create/choose TDB data directory, where Berkeley DB stores
   data and metadata.  Pass this to the daemon using the argument
   --tdb=DIRECTORY

*) Load user/password pairs into TDB, using tdbadm utility.  The user/password
   pairs are presented to tdbadm in a text file with this format:

	username <TAB> password <NL>

*) start the daemon!
